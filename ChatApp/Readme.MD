# Sample application for chat
Simple chat application developed in node js

## Pre-requisites
Node js needs to be installed [npm]

### Step 1
Install dependencies
npm install

### Step 2
Start the server
npm start

### Step 3
Launch browser http://localhost:3000

## Credits
